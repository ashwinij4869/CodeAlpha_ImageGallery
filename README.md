# 📸 Image Gallery

A modern, responsive, and interactive **Image Gallery** built using **HTML, CSS, and JavaScript**. The project offers a clean user interface with smooth animations, image filtering, and an immersive lightbox experience for viewing images.

## ✨ Features

* 📱 Fully responsive design for all screen sizes
* 🖼️ Interactive image gallery with modern grid layout
* 🔍 Lightbox view for enlarged image preview
* ⏮️ Previous & Next navigation controls
* 🎨 Smooth hover effects and CSS transitions
* 🗂️ Category-based image filtering
* ⚡ Fast, lightweight, and easy to customize

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
ImageGallery/
├── index.html
├── style.css
├── script.js

```

## 🚀 Getting Started

1. Clone or download this repository.
2. Open the project folder in Visual Studio Code.
3. Launch `index.html` in your browser or use the **Live Server** extension.
4. Browse the gallery, apply filters, and enjoy the lightbox experience.
